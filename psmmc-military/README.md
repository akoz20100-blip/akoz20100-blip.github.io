# PSMMC — Military Hospital Pharmacy Tools

This folder holds back-end tooling for the PSMMC pharmacy dashboard.

- [`packages/mcp-server`](packages/mcp-server) — a **remote MCP server** that lets anyone with
  Claude (desktop or phone) ask natural-language questions about the dashboard data
  (top withdrawals, planner expiry exposure, reorder pressure, …). See its
  [README](packages/mcp-server/README.md) for setup, deployment, and the Claude connector steps.
